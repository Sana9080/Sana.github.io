Vr-Rehabilitation

STACK: UNITY + C# + PYTHON

// MISSION BRIEFING

A Unity-built VR hand–arm cycling rehab experience for Meta Quest 3. The system streams real-time physiological data and adapts difficulty to keep users in the right training zone, improving engagement and usability for limited-mobility users.


// CORE SYSTEMS

> Real-time HR/HRV input → adaptive difficulty
> Game-based rehab loop → higher motivation than repetitive exercise
> Accessibility-first UI → usable in home or lab settings


// KEY IMPLEMENTATIONS

> Live physiological streaming + in-VR feedback (zones/alerts)
> Adaptive pacing algorithm (difficulty responds to the user)
> Optimized UI for clarity and low cognitive load
