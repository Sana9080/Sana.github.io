Festo-Machine-AR

STACK: UNITY + C# + REALVIRTUAL + AR

// MISSION BRIEFING

A Unity-built AR industrial training and visualization experience for the Festo machine. The app overlays an interactive digital twin onto the physical setup, using RealVirtual motion scripts to simulate workflows and present operational data in-context for faster learning, troubleshooting, and safer decision-making.


// CORE SYSTEMS

> AR machine overlay → component-level exploration + contextual info
> RealVirtual motion logic → Linear / Curve + Transport Guided + Guided Line
> Drive + Stopper behavior → Drive + Drive Erratic Position for controlled halts


// KEY IMPLEMENTATIONS

> Interactive hotspots → tap parts to reveal status, role, and workflow step
> Transport simulation → guided paths for object flow + station-to-station logic
> Fault/stop scenarios → stopper behavior to mimic jams, safety stops, and resets
